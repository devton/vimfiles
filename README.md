## Dependencies


Linux
```
apt-get install exuberant-ctags
```

Mac OSX
```
brew install ctags
```


## Install


```
git clone https://github.com/lerrua/vimfiles.git ~/.vim/
ln -s ~/.vim/vimrc ~/.vimrc
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
vim +BundleInstall +qall
```
